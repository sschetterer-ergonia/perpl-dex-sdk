# dex-sdk
Perpetual DEX SDK

## Prerequisites

* Rust >= 1.85.0
* `anvil` binary from [Foundry](https://getfoundry.sh/) for local testing


## Documentation

```
cargo doc --no-deps --open
```

## Usage

See [AionexTech/dex-sdk-examples](https://github.com/AionexTech/dex-sdk-examples) for some usage examples.
